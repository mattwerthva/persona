# NodeJs Boiler Service
This is a basic node.js boilerplate

### Setup
```
    git clone git@github.com:mattwerthva/node_boiler.git
    cd node_boiler
    nvm use 18.20.5
    npm install
```

### Generate OpenApi docs
```
    npm run docs
```

### Run service locally
```
    npm run start
```

### View OpenApi specs
```
   http://localhost:8000/swagger
```

### Run integration tests
```
   npm run test
```
